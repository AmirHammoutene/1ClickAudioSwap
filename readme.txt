----------
My program uses AudioEndPointController by Justin Ryder
https://github.com/justinryder/AudioEndPointController
I reworked it a little.
----------
My code is base on Qt 6.2.2
To compile code :
- Get Qt Creator (mine is 5.0.3 based on Qt 5.15.2), install for Qt6 with default modules (check if Qt5 compatibility is included)
https://www.qt.io/download
- Open 1ClickAudioSwap_src/1ClickAudioSwap.pro
- Compile

To optionally compile EndPointController :
- Get Microsoft Visual Studio
- Open 1ClickAudioSwap_src\EndPointController_reworked\EndPointController.sln
- Compile
